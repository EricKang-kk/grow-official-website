# GROW 官网开发路线图

最后更新：2026-05-17

## 1. 当前状态

项目处于正式开发前的准备阶段。

已经完成：

- 创建 GitHub 仓库
- 配置本地 Git 仓库
- 配置 GitHub SSH 访问
- 同步 AGENTS.md 项目说明文件
- 建立项目长期记忆文件结构

尚未开始：

- 页面和组件开发
- SEO、部署和上线

## 2. 开发原则

本项目是初学者学习型项目，优先选择清晰、稳定、容易理解的实现方式。

开发时遵守：

- 一个对话尽量只负责一个功能模块
- 一个小功能尽量在一台电脑上完成、提交、推送
- 换电脑开发前先执行 git pull
- 完成阶段性任务后提交并推送到 GitHub
- 重要背景、进度和决策写入项目文件，不只留在聊天里

## 3. 推荐对话拆分

建议至少使用以下对话：

1. 总控答疑对话
2. 项目初始化

## 4. 开发阶段

### 阶段 0：项目准备

状态：进行中

目标：

- 建立 GitHub 同步方式

完成标准：

- 两台 Mac 都能 clone / pull 仓库

### 阶段 1：项目初始化

状态：未开始

目标：

- 初始化 Next.js 项目
- 启用 TypeScript
- 启用 Tailwind CSS
- 建立基础目录结构
- 确认本地开发服务器可运行

建议产出：

- app/page.tsx

### 阶段 2：设计系统

建议产出：
